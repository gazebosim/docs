◆ UpdateConfigAttribute()

bool UpdateConfigAttribute ( const std::string &  _path,
const std::string &  _attribute,
const bool  _value 
) const

Update an attribute on an XML file. The attribute belongs to a <dialog> element that has a name attrbute matching this dialog's name, i.e.

<dialog name="dialog_name" attribute="value"/>

If a dialog element with this dialog's name doesn't exist yet, one will be created.

Parameters
[in]_pathFile path. File must already exist, this function will not create a new file.
[in]_attributeXMLElement attribute name
[in]_valueXMLElement attribute value
Returns
True if written to config file
