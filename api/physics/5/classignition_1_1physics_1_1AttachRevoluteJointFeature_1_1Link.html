Attach this link to another link using a revolute joint.

Parameters
[in]_parentThe parent link for the joint. Pass in a nullptr to attach the link to the world.
[in]_nameThe name of this joint.
[in]_axisThe joint axis for the new joint. The rest of the joint properties will be left to the default values of the physics engine.
Returns
A reference to the newly constructed RevoluteJoint.
