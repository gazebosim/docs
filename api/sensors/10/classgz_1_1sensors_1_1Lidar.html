Detailed Description

Lidar Sensor Class.

This class creates laser scans using. It's measures the range from the origin of the center to points on the visual geometry in the scene.

It offers both a gz-transport interface and a direct C++ API to access the image data. The API works by setting a callback to be called with image data.
