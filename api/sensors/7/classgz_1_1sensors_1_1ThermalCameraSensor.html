Detailed Description

Thermal camera sensor class.

This class creates thermal image from a Gazebo Rendering scene. The scene must be created in advance and given to Manager::Init(). It offers both a gz-transport interface and a direct C++ API to access the image data. The API works by setting a callback to be called with image data.
