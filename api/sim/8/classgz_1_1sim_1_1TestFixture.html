Detailed Description

Helper class to write automated tests. It provides a convenient API to load a world file, step simulation and check entities and components.

Usage

// Load a world with a fixture
TestFixture fixture("path_to.sdf");
// Register callbacks, for example:
fixture.OnPostUpdate([&](const sim::UpdateInfo &,
const sim::EntityComponentManager &_ecm)
{
// Add expectations here
}).Finalize();
// Be sure to call finalize before running the server.
// Run the server
fixture.Server()->Run(true, 1000, false);

TestFixture& Finalize ( )
