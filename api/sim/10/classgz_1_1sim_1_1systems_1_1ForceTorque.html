Detailed Description

This system manages all Force-Torque sensors in simulation. Each FT sensor reports readings over Gazebo Transport.

Note
Regardless of the setting of //sensor/force_torque/frame the point of application of the force is at the sensor's origin. //sensor/force_torque/frame only changes the coordinate frame in which the quantities are expressed, not the point of application.
