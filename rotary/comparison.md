# Feature comparison

Rotary is Gazebo's rolling release stream, built continuously from the
`main` branches of the Gazebo libraries. Because it changes every day,
it does not have a fixed feature matrix in the same way as a named
release.

For the latest stable comparison against Gazebo Classic, use the
[Jetty feature comparison](https://gazebosim.org/docs/jetty/comparison/),
which reflects the newest named Gazebo release.

If you are evaluating unreleased work in Rotary, also check:

- [Release Features](release-features) for features tracked by release
- [About Rotary](rotary) for packaging and installation details
- the relevant library API references under this release's
  **Library Reference** section
